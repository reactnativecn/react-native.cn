渲染一个Boolean输入

这是一个“受控组件”（controlled component）。你必须使用onValueChange回调来更新value属性以响应用户的操作。如果不更新value属性，组件只会按一开始给定的value值来渲染且保持不变。

注意:CheckBox只在Android端实现，IOS端暂未实现

@keyword checkbox @keyword toggle
